# Sistema-Agendamento-de-Salas
Sistema corporativo de agendamento de salas no Discord em Node.js. Permite reservar com validação de data, horário e conflitos, escolher participantes, enviar convites, confirmar presença, enviar lembretes automáticos, cancelar com segurança, visualizar em calendário, exportar para Excel e enviar relatórios por e-mail.
